# VisionCraft fork marker

This directory is a **vendored VivecraftMod** tree modified for VisionCraft (Mac → Vision Pro).

Build Fabric mod from repo root:

```bash
cd minecraft/VivecraftMod
./gradlew :fabric:build
```

Default VR plugin: **Apple Vision**. Start `mac-host/VisionCraftHost` before enabling VR in Minecraft.

See `/minecraft/VENDORED.md` for upstream pin and merge policy.
